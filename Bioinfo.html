去除线粒体、叶绿体free DNA：Samtoools
for i in *sam
do
     samtools view -@ 6 -bS $i > ${i/sam}bam

Peak caller: MACS2 callpeak
for i in *shift.sort.bam
do
     macs2 callpeak -t $i -f BAMPE -g 1.2e8 --keep-dup all --cutoff-analysis -n ${i/.shift.sort.bam} --outdir ../07.peak/${i/.sort.bam} 2> ../06.log/${i/sort.bam}macs2.log.txt
done > ../06.log/jim.shift.macs2.log.txt 2>&1

-t –treatment FILENAME: input file direction

-c –control: the control, genomic input or mock IP data file

-n –name: the name of experiment

–outdir: MACS will save all output files into the specified folder for this option

-f –format FORMAT: BAMPE and BEDPE need to be specified

-g –gsize: it’s the mappable genome size or effective genome size which is defined as the genome size which can be sequenced. 拟南芥实际基因组大小1.2e8（一亿两千万）

–keep-dup: It controls the MACS behavior towards duplicate tags at the exact same location
